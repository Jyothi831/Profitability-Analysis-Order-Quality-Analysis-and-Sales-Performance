# Profitability-Analysis-Order-Quality-Analysis-and-Sales-Performance
Developed an end-to-end Business Performance Analytics solution using Power BI, SQL, and Excel to analyze profitability, sales trends, and operational quality metrics. Extracted and transformed raw data using SQL and Power Query, and created calculated measures using DAX to track KPIs such as revenue, profit margins, order volume, return rate, defect percentage, and regional growth.

Designed interactive dashboards with drill-through, slicers, and KPI indicators to compare monthly, quarterly, and regional performance. Identified high-margin products, loss-making segments, and quality-related order issues, enabling data-driven business decisions and strategic performance improvement.
